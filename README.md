# Farmer's Delight: Plus

### Overview

**Farmer's Delight: Plus** is an add-on that expands the cooking system introduced in Farmer's Delight mod for Minecraft.

The mod adds new **meals**, new types of dough and noodles, along with a variety of **snacks** and **drinks**. You can try different recipes: **rice with honey and berries**, **pancakes** with various fillings, and more! It also introduces new **feasts** to decorate and serve.

Everything is designed in a clean **vanilla style** and balanced to fit naturally into the game. Meals can be prepared using a crafting table or the Farmer's Delight cooking pot, following the familiar **cooking mechanics**.

Farmer's Delight: Plus is compatible with mods like **Farmer’s Respite**, **Autumnity** and **Create**. These are not required, but they can expand the overall experience.
